Uso de Robocom Web

La página web es compatible con Google Chrome 78.0.3904.97, Mozilla Firefox 70.0, Opera 64.0.3417.61

Para acceder a ella, debe utilizarse cualquier dispositivo (computador o celular) que cuente con tecnología Wi-Fi y un navegador web.
El procedimiento para conexión debe ser el siguiente:

1- Encender el módulo ESP8266.
2- Encender el brazo robot.
3- Desconectarse de cualquier otra red (móvil, ethernet, WiFi) ya que no podrá utilizar la página web si posee otra conexión. 4-Establecer la conexión a través de un dispositivo compatible. Como se indica a continuación, desde un computador con Windows 10:


5- Ingresar usando SSID: "ESP8266-WiFi" y contraseña WPA/WPA2 PSK: "esp123456".
6- Desde el navegador, escribir la dirección "192.168.4.1:8266". Si la dirección es correcta, se encenderá un LED desde el módulo de acceso remoto que se apagará cuando la página cargue completamente.

Probador


Esta sección sirve para probar algunas trayectorias sin retardo.

En el campo de texto el usuario debe escribir cada línea de comando de acuerdo a la siguiente estructura:

qi[ESPACIO]AAA.BB
[ENTER]
qi[ESPACIO]AAA.BB

En donde:
i: 0,1,2,3,4,5,6 representa a cada uno de los motores del brazo manipulador, siendo los motores A, B, C, D, E, F respectivamente.

AAA: Indica la cantidad de dígitos máxima que pueden escribirse para la parte entera del valor, el cuál debe escribirse en grados.
BB: Indica la cantidad de dígitos máxima que debe tener siempre la mantisa. Esto es, si se trata de un número entero, debe escribirse AAA.00.
Al culminar de escribir los comandos deseados, debe presionar el botón "Pulsar" para que el robot comience el movimiento.

NOTA: Se recomienda no usar una gran cantidad de comandos en el probador ya que podrán apreciarse las últimas instrucciones y no de la forma que desea el usuario. Es decir, estructuradamente.

Configuración de parámetros

En esta sección el usuario podrá configurar las constantes asociadas a los controladores PID de los motores A, B y C. Además, podrá habilitarse o deshabilitarse los puentes H de los respectivos motores.
Por otra parte, podrán configurarse las constantes de calibración y el registro del temporizador que utiliza el microcontrolador dsPIC30F3011 para gestionar el controlador PID.

NOTA: Se recomienda al usuario no manipular las constantes de calibración ni del registro del temporizador si no está estrictamente seguro de los valores, ya que puede generar desperfectos en el funcionamiento del robot, pudiendo culminar en daños severos a la integridad del equipo.
