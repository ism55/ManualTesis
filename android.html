Uso de Robocom Android

La aplicación debe estar instalada en un dispositivo Android con sistema operativo Oreo (8.1) o superior.

Robocom Android, requiere que el dispositivo deba conectarse a la señal Wi-Fi generada por el módulo. De otro modo, la aplicación generará un error. Para conectarse, debe encender el módulo ES8266 y buscar la señal identificada con el nombre SSID "ESP8266-WiFi", con contraseña WPA/WPA2 PSK: "esp123456".

Número de repeticiones

El panel "Número de repeticiones" sirve para que el usuario seleccione el número de veces que desea ser repetida la lista de comandos. Por defecto el valor es "0" para que se ejecute solamente una vez.



Status


El panel "Status" se utiliza para visualizar el status de la conexión. En particular, para el envío de tramas.
Si Status indica "200", la comunicación fue exitosa. En cambio, si indica "404" indicará que el envío fue incorrecto o que no hubo comunicación.



Escribir comandos


En el campo de texto que indica "Ingrese comando cmd xxx y presione enter" el usuario debe escribir cada línea de comando de acuerdo a la siguiente estructura:

qi[ESPACIO]AAA.BB
[ENTER]
qi[ESPACIO]AAA.BB

En donde:
i: 0,1,2,3,4,5,6 representa a cada uno de los motores del brazo manipulador, siendo los motores A, B, C, D, E, F respectivamente.

AAA: Indica la cantidad de dígitos máxima que pueden escribirse para la parte entera del valor, el cuál debe escribirse en grados.
BB: Indica la cantidad de dígitos máxima que debe tener siempre la mantisa. Esto es, si se trata de un número entero, debe escribirse AAA.00.
Al culminar de escribir los comandos deseados, debe presionar el botón "Iniciar" para que el robot comience el movimiento.

El botón "Cargar Archivo" se utiliza para seleccionar el archivo de texto en formato *.txt, el cual debe estar escrito con la misma estructura anteriormente descrita.



Guardar lista de comandos



Si el usuario desea guardar la lista de comandos que escribió en el campo de texto, puede hacerse presionando el botón "Guardar a archivo" luego de indicar el nombre del archivo correspondiente. Esto generará un fichero de texto con extensión *.txt que puede ser utilizado nuevamente desde la opción "Cargar archivo".

Es necesario mencionar que no podrá generarse un nuevo archivo si no indica el nombre de este.
